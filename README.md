# react-native-kakao-map-v2

KakaoMapSDKv2 integration for react native

## Installation

```sh
npm install react-native-kakao-map-v2
```

## Usage

```js
import { KakaoMapV2View } from "react-native-kakao-map-v2";

// ...

<KakaoMapV2View color="tomato" />
```
